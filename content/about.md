# About Us

Welcome to our Hugo Tailbliss site! This is a modern, fast, and responsive website built with Hugo and the Tailbliss theme.

## Features

- Built with Hugo static site generator
- Styled with Tailwind CSS via the Tailbliss theme
- Responsive design that works on all devices
- Fast loading and optimized for performance

## Technology Stack

- **Hugo**: Fast static site generator
- **Tailbliss Theme**: Modern Hugo theme with Tailwind CSS
- **Tailwind CSS**: Utility-first CSS framework
- **Alpine.js**: Lightweight JavaScript framework

This site demonstrates the power and flexibility of modern static site generators combined with utility-first CSS frameworks.
